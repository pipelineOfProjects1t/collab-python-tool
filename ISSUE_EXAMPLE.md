# Issue example

## Заголовок

Добавить проверку количества участников

## Описание

В проекте «Учёт мероприятий» нужно добавить проверку пользовательского ввода и защиту от некорректных данных.
Это позволит сделать приложение надежнее и показать работу с GitHub Issues.

## Как закрыть Issue коммитом

```bash
git commit -m "Добавить проверку количества участников. Closes #1"
```

После пуша ветки на GitHub задача закроется автоматически, если номер Issue действительно равен `#1`.
