# Информация о шаблоне

**Номер проекта:** 30

**Название папки:** `collab-python-tool`

**Шаблон проекта:** `Учёт мероприятий`

**Ссылка на GitHub:** `https://github.com/pipelineOfProjects1t/collab-python-tool`

**Файл данных:** `events.json`

## Что лежит в папке

- `main.py` — готовое консольное приложение на Python.
- `events.json` — JSON-файл для хранения данных.
- `README.md` — описание проекта, запуск, ссылка на GitHub и порядок сдачи.
- `GIT_STEPS.md` — пошаговые команды Git и GitHub.
- `ISSUE_EXAMPLE.md` — пример задачи в GitHub Issues.
- `PULL_REQUEST_DESCRIPTION.md` — пример описания Pull Request.
- `tests/` — модульные тесты.

## Какой это шаблон

Это шаблон проекта «Учёт мероприятий». Его можно использовать как отдельный репозиторий `collab-python-tool` для практического кейса по совместной разработке с Git и GitHub.
