## Описание

Это готовое решение для практического кейса по совместной разработке с Git и GitHub.
Проект представляет собой консольное приложение для планирования мероприятий, мест, дат и участников.
Приложение написано на Python и работает через консольное меню.
Данные хранятся в файле `events.json`.

## Меню приложения

1. Добавить запись
2. Показать все записи
3. Показать статистику
4. Удалить запись
5. Выход

## Поля записи

- `title` — Название мероприятия
- `date` — Дата
- `place` — Место
- `participants` — Количество участников

## Как запустить

```bash
python main.py
```

## Как запустить тесты

```bash
python -m unittest discover
```

## Ветки для разработки

- `feature/add-record` — добавление записи
- `feature/list-and-stats` — просмотр списка и статистика
- `feature/delete-record` — удаление записи
